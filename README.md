# 웹프로그래밍 과제

### html과 css를 이용하여 이력서 페이지 만들기

- 시맨틱 태그 : 3강

  - `<header>`
  - `<nav>`
  - `<section>`
    - `<article>`
  - `<footer>`
  - `<form>`

- 요소 : 2강

  - 오디오
  - 리스트
  - 이미지
  - 목차(본문에 앵커와 연결)
  - 링크(클릭하면 새 창이나 새 탭에 출력)

- css : 4강

  - 배경색
  - 테두리(색, 둥글게)
  - 폰트(종류, 크기)
  - 들여쓰기
  - `<hr>`

- css 배치
  - `<header>`, `<nav>`, `<section>`, `<footer>` 등의 영역을 적절히 배치하기
